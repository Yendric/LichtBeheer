Setup

Om de applicatie te laten werken, moet er een API-gebruiker aangemaakt worden op uw Philips Hue bridge. Omwille van browser mixed content policies is dit proces redelijk ingewikkeld, maar u hoeft het maar eenmaal uit te voeren.

Ten eerste moet u het SSL-certificaat van uw lokale philips hue api vertrouwen. Ga hiervoor naar {{lampService.hueApiUrl}}, klik op 'Geavanceerd' en 'Toch doorgaan' en sluit het tabblad. Nu kan onze website communiceren met de hue api op je netwerk.

U heeft 60 seconden de tijd om op de knop op uw philips hue bridge te drukken.

Vervolgens dient u op onderstaande knop te drukken.
